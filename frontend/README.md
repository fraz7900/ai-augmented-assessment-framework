# frontend/

React application: document upload UI, evidence review workflow (accept/edit/reject AI-proposed mappings), maturity scoring dashboard, and report download.

Not scaffolded with a specific build tool yet (Vite vs. Create React App vs. Next.js is a Sprint 6 decision, made when the dashboard work actually begins) to avoid committing to frontend tooling before there is a backend API contract to build against. See `docs/architecture/00-repository-architecture.md`.
